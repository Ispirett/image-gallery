# Image Gallery Rails Application

[![Greenkeeper badge](https://badges.greenkeeper.io/cmckni3/image-gallery.svg)](https://greenkeeper.io/)

## Requirements

* ImageMagick
* Ruby 2.1.2
* Rails 4.1.4

## Setup

* Copy `.env` file

```bash
cp .env.example .env
```

* Change the `.env` variables

* Bundle install

```bash
bundle install
```

* Install foreman

```bash
gem install foreman
```

* Run foreman

```bash
foreman start
```
